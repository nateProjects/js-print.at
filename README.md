# print.At

Basic Canvas Text Print Library

Adds a Print.At command to JavaScript

For a future Pseudo-Basic project

**Usage:**
*(Within a HTML web page)*

* screen.font ("Monaco",12,4);
* screen.size (320,240);
* print.at (x,y,"text");
* screen.clear ();
